# Organizador de Arquivos

Organizador automático de arquivos por tipo, com interface gráfica em Python (tkinter).

## Funcionalidades
- Escolha a pasta via interface
- Cria pastas por extensão de arquivo
- Move arquivos automaticamente
- Mostra total de arquivos organizados
- Executável `.exe` pronto (Windows)

## Como usar
1. Abrir `main.py` com Python 3.x
2. Ou usar o `.exe` em `dist/Nordio.exe`
3. Clicar no botão "Escolher Pasta e Organizar"

## Tecnologias
- Python
- tkinter
- os, shutil
